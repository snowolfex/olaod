# Free 3D Llama/Alpaca References

These are source previews for a free, reusable llama/alpaca direction that feels adjacent to an Ollama-style mascot without copying Ollama branding directly.

## Best fit

- `free-lowpoly-llama-ccby-preview.jpg`
  - Best immediate match for a simple mascot direction.
  - Source: https://sketchfab.com/3d-models/llama-2b7ca83db72a4ff6898869e1983872e3
  - Title: `llama`
  - Creator: `niraldnaik`
  - License: `CC BY 4.0`
  - Notes: Low-poly, very easy to simplify into a head/bust icon.

- `free-lowpoly-alpaca-ccby-preview.jpg`
  - Best stylized option if you want softer, friendlier proportions.
  - Source: https://sketchfab.com/3d-models/low-poly-alpaca-4d9326a1d9054bb9852ba32b28b09b4b
  - Title: `Low poly alpaca`
  - Creator: `TinyGabe`
  - License: `CC BY 4.0`
  - Notes: Cartoonish silhouette; easiest to adapt into a playful 3D head.

- `free-alpaca-ccby-preview.jpg`
  - Best if you want something fuller and more polished.
  - Source: https://sketchfab.com/3d-models/alpaca-a81fdf43bbec4d7baae161cb63a11823
  - Title: `Alpaca`
  - Creator: `kenchoo`
  - License: `CC BY 4.0`
  - Notes: More detailed than the low-poly options.

## Recommended direction

If the goal is "3D kinda Ollama head", start from `free-lowpoly-alpaca-ccby-preview.jpg` or `free-lowpoly-llama-ccby-preview.jpg` and make an original bust/head render with:

- rounded muzzle
- tall upright ears
- chunky wool mass
- clean matte clay shading
- no direct reuse of Ollama marks, exact facial proportions, or logo geometry

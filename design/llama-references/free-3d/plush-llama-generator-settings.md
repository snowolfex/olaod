# Plush Llama Generator Settings

## General target

- clean premium mascot render
- high shape readability
- soft plush material response
- centered composition first

## Suggested settings for most image generators

- aspect ratio: `1:1` for mascot studies
- aspect ratio: `4:5` for portrait splash studies
- guidance / prompt adherence: medium-high
- stylization: medium
- detail: medium
- contrast: soft-medium
- camera: slight telephoto or neutral portrait feel

## Good starting values

If the tool exposes these controls, start around here:

- steps: `28-40`
- cfg / guidance: `6-8`
- stylize: moderate
- sharpness: low-medium
- chaos / variation: low for first passes, medium for exploration

## Iteration order

1. Lock silhouette first.
2. Lock palette second.
3. Add fabric and seam detail third.
4. Add accessories only after the base mascot works.

## Best first-pass setup

- square frame
- centered bust or full mascot
- plain soft background
- one plush llama only
- no extra props

## When results drift wrong

If it becomes too realistic:

- add `stuffed-animal`, `plush mascot`, `toy proportions`, `simple readable forms`

If it becomes too childish or cheap:

- add `premium`, `tasteful`, `clean 3d illustration`, `soft luxury palette`

If it becomes too pink or candy-like:

- add `warm cream`, `oat`, `cocoa`, `dusty teal`, `no pink tones`

If it becomes too plastic:

- add `velvety fabric texture`, `matte plush surface`, `soft fibers`, `no gloss`

## Best negative prompt block

`pink, hot pink, magenta, realistic fur, plastic, vinyl, latex, toy shine, angry, evil, horror, photoreal, sharp edges, hard shadows, thin face, long neck, cheap cartoon, clutter, copied logo`

# OL Icon Art Direction

The earlier mascot lane is retired. All shipped brand art should now use the OL monogram.

## Core shape

- Build a rounded O ring with a warm premium gradient.
- Lock the L into the right side of the O so it reads as one mark, not two floating letters.
- Keep the silhouette bold enough to survive favicons, installer icons, PDF export callouts, and admin status cards.

## Finish

- Light theme: warm cream-to-copper finish with a soft gloss.
- Tech theme: preserve the same base form and add cooler signal highlights instead of changing the silhouette.
- Dark theme: keep the copper core but move the field and shadow treatment into a deeper midnight range.

## Usage rules

- Prefer the monogram over mascots, character art, or illustrative animals.
- In docs and help surfaces, use one hero mark plus simple interface framing instead of decorative scene art.
- For PDFs, section illustrations should use the same OL mark language with small accent changes only when the section needs extra distinction.

## Avoid

- No animal or mascot-driven graphics.
- No split O and L layouts that weaken recognition.
- No over-detailed rendering that breaks down at small sizes.

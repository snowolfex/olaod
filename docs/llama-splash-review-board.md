# Llama Splash Review Board

Use this sheet for fast approval decisions.

Related full brief:
- See `docs/llama-splash-art-direction.md` for the full visual, motion, and prompt notes.

Decision keys:

- `Approved`
- `Needs revision`
- `Rejected`

Recommended review pass:

1. Pick the strongest visual in each theme first.
2. Mark any weak candidate as `Needs revision` instead of trying to force approval.
3. Keep all approved llamas in the same family silhouette.
4. Do not approve a variant that only works at desktop size.

## Random Splash Rule After Approval

Once each theme has three approved visuals:

1. Read active theme.
2. Select that theme's approved pool.
3. Randomly pick one candidate.
4. Avoid the immediately previous llama for the same theme if possible.
